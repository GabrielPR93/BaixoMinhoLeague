# BaixoMiño League

Aplicación Android desarrollada en Kotlin para organizar eventos como campeonatos o ligas y gestionarlas.
Ejemplo: Futbolin,Billar,Tenis...

>Librerías

- Firebase:
  - Analytics
  - Authentication
  - Firestore Database
  - Storage
  - Realtime Database
  
- Picasso
- Navigation Component
- Material Design
- Dagger hilt
